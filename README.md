0. Data
   - We open available data through the link in our Google Drive:
    https://drive.google.com/drive/folders/1XzebWMcxl0V5cGGDnXD0imPetLikjN5T?usp=sharing

2. System requirements
   - Python 3 should be enough, we recommend to use Google Colaboratory, with it, you do not need any other software.
   - There is no required non-standard hardware.

3. Installation guide
   - Log-in Google Colaboratory and connect the data folder with your Google Drive folder to use the data we have shared with the link above.
   - There is no other installation needed except running the codes.

4. Demo
   - Mount your Google Drive, and run the codes following the order.
   - The output(Figures) will be shown on the Google Colab page you are working on.
   - Expected run time for demo on a normal desktop computer is less than 30 minutes.
